# Bachire Shop

This is the initial setup for the Bachire Shop, an e-commerce platform.

## Project Structure
- `frontend`: Contains the React application.
- `backend`: Contains the Node.js/Express API.
- `README.md`: Provides documentation for the project.
- `.gitignore`: Specifies files and directories that should be ignored by Git.

## Getting Started

### Prerequisites
- Node.js
- NPM
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/bachiroud321-crypto/bachire-shop.git

# Navigate into the directory
cd bachire-shop

# Install dependencies
# For frontend
cd frontend
npm install

# For backend
cd ../backend
npm install
```

### Running the Application

```bash
# Start the frontend
cd frontend
npm start

# Start the backend
cd backend
node index.js
```
